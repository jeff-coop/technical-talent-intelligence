# Software Engineering

This page is my attempt to better understand software engineering, the different types of software engineers, and the environments where exceptional software engineering talent develops.

Software engineering sits at the center of modern technology organizations and overlaps with infrastructure, AI, security, data, product, and platform engineering.

The goal is not to become a software engineer.

The goal is to better understand the skills, experiences, and technical ecosystems that shape strong engineering talent.

---

# What Is Software Engineering?

Software engineering is the practice of designing, building, testing, deploying, and maintaining software systems.

Software engineers solve problems using technology.

Those problems can range from:

* Consumer Applications
* Enterprise Software
* Infrastructure Platforms
* Artificial Intelligence
* Financial Systems
* Developer Tools
